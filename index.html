MOBEXLER - A Mobile Application Penetration Testing Platform

Mobexler is a customised virtual machine, based on Elementary OS , designed to help in penetration testing of Android & iOS applications.

Download →

Awesome tools

Mobexler comes preinstalled with various open source tools, scripts, prerequisites etc. which are needed for security testing Android & iOS application.

Android Zone

Contains the various updated tools only for Android Security Testing

General Penetration Testing Zone

Contains the various updated tools only for IOS Penetration testing

IOS Zone

Contains the tools that useful for general Penetration testing like Nmap,Metasploit etc

Unique

Mobexler is unique for several reasons

Better manage

One platform capable of managing Android and IOS devices at the same time .

Free

It is free for everyone to use

Various tools

Comes with preinstalled different updated tools

Docker supported

Tools like MobSF run inside a docker container and ensures that it is easier to download the newer versions

Static Analysis on IPA

MobSF in container also gives capability to perform Static analysis on IPA,without Xcode tools

Easy To Connect

Android/iOS devices can easily be connected through USB and various tools can be used to perform the testing, like Frida, Objection, Hopper etc.
